# PyHacks

PyHacks is a Command-line based hacking-tools library written in Python.

## Installation

Clone the repository and use:

```bash
python3 main.py
```

to enter the command-line-interface.

## Usage

In case of usage as a package, you may import it like so:

```python
import pyhack

...
...
...
```

*Please Note that the package version of this library is still under development*
